# Post Title

## Usage

```js
import { PostTitle } from '@10up/block-components';

function BlockEdit() {

    return (
        <PostTitle tagName="h2" className="wp-block-example-hero__title" />
    )
}
```

## Props

| Name       | Type              | Default  |  Description                                                   |
| ---------- | ----------------- | -------- | -------------------------------------------------------------- |
| `context` | `object` | `{}` |  |
| `tagName` | `string` | `h1` |  |
| `...rest` | `object` | `{}` |  |
